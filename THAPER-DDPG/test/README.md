# Test checkpoints

This folder is reserved for pretrained checkpoints used by `test.py`.

Due to file size limits, the checkpoint files are not uploaded directly to this repository.

Please download the test checkpoints from the following Baidu Netdisk link:

Link: [test](https://pan.baidu.com/s/1H7TIytFURLJTLsvaKRgT_g)  
Extraction code: xyys

After downloading, place the files into the following structure:

test/
├─ Drone1/
├─ Drone2/
└─ Drone3/

Each subfolder should contain the corresponding checkpoint files required by the code, such as:
- checkpoint
- model.index
- model.meta
- model.data-00000-of-00001
