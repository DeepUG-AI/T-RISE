# Test checkpoints

This folder is reserved for the pretrained checkpoint files used by `test.py`.

These files store the trained model parameters for the three UAV agents and are required for evaluation/testing.  
Without them, `test.py` cannot reproduce the trained policy correctly.

Due to file size limits, the checkpoint files are not uploaded directly to this repository.

Please download them from the following Baidu Netdisk link:

Link: [test](https://pan.baidu.com/s/1H7TIytFURLJTLsvaKRgT_g)  
Extraction code: xyys

Please place the downloaded files under the `test` directory before running `test.py`.
